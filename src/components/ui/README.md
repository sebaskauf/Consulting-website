# UI Components from 21st.dev

This directory is for UI components copied from [21st.dev](https://21st.dev).

## How to use 21st.dev components with Claude Code

### Method 1: Direct Copy-Paste (Fastest)
1. Browse components on [21st.dev](https://21st.dev)
2. Copy the component code
3. Ask Claude: "Create a new component file here for [component name]" and paste the code
4. Import and use it in your pages/components

### Method 2: Claude Code Integration (Recommended)
1. Browse components on [21st.dev](https://21st.dev)
2. Tell Claude: "Add the [component name] from 21st.dev to my project"
3. Provide the component URL or code
4. Claude will:
   - Create the component file in the right location
   - Fix any imports/dependencies
   - Show you how to use it

### Example Components to Try
- Hero sections
- Feature grids
- Pricing tables
- Testimonials
- CTAs (Call to Actions)
- Navigation bars
- Footers

## File Naming Convention
Use descriptive names in kebab-case:
- `hero-section.tsx`
- `feature-grid.tsx`
- `pricing-card.tsx`
